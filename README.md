# SPA github-app

Simple application for fetching public repositories and forks of selected GitHub user.

## Usage

Home page displays list of GitHub user's repositories/forks. It is also available to view detailed info on separate route, which also includes readme file. 
Link to GitHub repository itself is also included.

## Technologies
* Vue, vue-router
* Pinia
* axios
* Bootstrap, SASS
* dayjs - library for date/time formatting
* marked - library for markdown formatting

## How to use
App is not deployed, you can run it locally with instructions below:

1. clone or download this repository
2. open directory in terminal, run `npm install` command
3. run the app with `npm run dev` command
